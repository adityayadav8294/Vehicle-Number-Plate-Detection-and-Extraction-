

# Vehicle Number Plate Detection and Extraction

This project uses **YOLOv8** and **EasyOCR** to detect and read vehicle number plates from a video.

---

## 🛠 Features

✔ Detects license plates in videos  
✔ Tracks vehicles & plates  
✔ Extracts text using OCR  
✔ Saves annotated output video
