# 🚗 Vehicle Number Plate Detection and Extraction

A deep learning–based project for detecting and extracting vehicle number plates from video input using YOLO-based object detection and tracking.

---

## 📌 Project Overview

This system processes vehicle video footage to:

- Detect license plates using a trained YOLO model
- Track detected plates across frames
- Generate processed output video with detections
- Support modular tracking architecture

Designed for smart surveillance, traffic monitoring, and parking systems.

---

## 🛠 Tech Stack

- Python 3.x
- YOLO
- OpenCV
- NumPy
- Custom Tracking Module

---

## 📂 Project Structure

Vehicle-Number-Plate-Detection-and-Extraction/

- input/ (Add your own test video here)
- my_tracking/
- utils/
- main.py
- requirements.txt

## ⚙️ Installation & Setup

### Clone Repository
git clone https://github.com/adityayadav8294/Vehicle-Number-Plate-Detection-and-Extraction-

### Create Virtual Environment
python -m venv venv

### Activate Environment (Windows)
venv\Scripts\activate

### Install Dependencies
pip install -r requirements.txt

---

## ▶️ Run the Project

python main.py

📌 Add your own video file inside the `input/` folder before running.
